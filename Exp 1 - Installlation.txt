Experiment No: 1
Title: Installation and Configuration of Flutter Environment (Linux)

Aim:
To install and configure Flutter environment on Linux system.

Requirements:

* Ubuntu/Linux OS
* Internet connection
* VS Code

---

### Step 1: Update System

Open terminal and run:

sudo apt update
sudo apt upgrade -y

---

### Step 2: Install Required Tools

Run:

sudo apt install git curl unzip xz-utils zip libglu1-mesa -y

---

### Step 3: Download Flutter SDK

Run:

cd ~
git clone https://github.com/flutter/flutter.git

---

### Step 4: Set PATH

Run:

export PATH="$PATH:$HOME/flutter/bin"

To make permanent:

echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bashrc
source ~/.bashrc

---

### Step 5: Verify Installation

Run:

flutter --version

---

### Step 6: Check Setup

Run:

flutter doctor

---

### Step 7: Install Android SDK (Optional)

Install Android Studio:

sudo snap install android-studio --classic

Open Android Studio and install:

* Android SDK
* Emulator

---

### Step 8: Accept Licenses

Run:

flutter doctor --android-licenses

Type:
y

---

### Step 9: Fix Errors

Run again:

flutter doctor

Ensure all checks are complete

---

### Step 10: Create Flutter Project

Run:

flutter create my_app
cd my_app

---

### Step 11: Run Application

Run:

flutter run

---

### Output:

Flutter environment is successfully installed and application runs on Linux system.

---

### Conclusion:

Flutter environment is successfully configured on Linux and ready for application development.
