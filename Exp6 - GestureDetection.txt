To apply concepts of gestures using GestureDetector widget in Flutter.

7. Paste the below code

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Smart Lock',
      debugShowCheckedModeBanner: false,
      home: const SmartLockScreen(),
    );
  }
}

class SmartLockScreen extends StatefulWidget {
  const SmartLockScreen({Key? key}) : super(key: key);

  @override
  _SmartLockScreenState createState() => _SmartLockScreenState();
}

class _SmartLockScreenState extends State<SmartLockScreen> {
  bool isLocked = true;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text("Smart Door Lock"),
        backgroundColor: isLocked ? Colors.red : Colors.green,
      ),
      body: AnimatedContainer(
        duration: const Duration(milliseconds: 500),
        decoration: BoxDecoration(
          gradient: LinearGradient(
            colors: isLocked
                ? [Colors.red.shade100, Colors.grey.shade100]
                : [Colors.green.shade100, Colors.grey.shade100],
            begin: Alignment.topCenter,
            end: Alignment.bottomCenter,
          ),
        ),
        child: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Icon(
                isLocked ? Icons.lock : Icons.lock_open,
                size: 120,
                color: isLocked ? Colors.red : Colors.green,
              ),

              const SizedBox(height: 50),

              GestureDetector(
                onTap: () {
                  setState(() {
                    isLocked = !isLocked;
                  });
                },
                child: Container(
                  padding: const EdgeInsets.symmetric(
                      horizontal: 40, vertical: 15),
                  decoration: BoxDecoration(
                    color: isLocked ? Colors.red : Colors.green,
                    borderRadius: BorderRadius.circular(30),
                  ),
                  child: Text(
                    isLocked ? "TAP TO UNLOCK" : "TAP TO LOCK",
                    style: const TextStyle(
                      color: Colors.white,
                      fontSize: 18,
                      fontWeight: FontWeight.bold,
                    ),
                  ),
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

8. Click "Run"

Output:
A Smart Lock app will be displayed. On tapping button, lock icon changes between locked and unlocked state. 

Conclusion:
In this experiment, GestureDetector is used to detect tap events and update UI. 


Description:

In this experiment, gesture handling was implemented in a Flutter application using the GestureDetector widget on the FlutLab platform. The objective was to understand how user interactions, specifically touch gestures, can be detected and used to trigger dynamic changes in the user interface.

The application simulates a Smart Door Lock system, where the lock state (locked/unlocked) is controlled through user input. The GestureDetector widget is used to capture tap events, which toggle the state of the lock. This interaction is managed using a StatefulWidget, allowing the UI to update in real time based on changes in the application state.

To enhance the visual experience, the app incorporates AnimatedContainer, which smoothly transitions between different UI states, including background gradients and color changes. Icons and text dynamically update to reflect the current lock status, providing immediate feedback to the user.

This experiment demonstrates how gesture recognition, state management, and animation can be combined to create responsive and interactive applications. It highlights the importance of user input handling in modern mobile app development.
