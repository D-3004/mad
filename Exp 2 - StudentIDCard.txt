Title: Student ID Card UI using Flutter

Aim:
To develop a Flutter UI for creating a Student ID card using basic widgets. 

Requirements:

* FlutLab account
* Internet

Steps:

1. Open https://flutlab.io

2. Click on "New Project"

3. Enter project name: id_card_app

4. Click "Create Project"

5. Go to lib/main.dart

6. Delete existing code

7. Copy and paste the following code:

import 'package:flutter/material.dart';

void main() {
  runApp(const MaterialApp(
    debugShowCheckedModeBanner: false,
    home: IdCard(),
  ));
}

                ),
                child: Column(
                  mainAxisAlignment: MainAxisAlignment.center,
                  children: const [
                    Icon(Icons.account_balance, color: Colors.white, size: 30),
                    SizedBox(height: 5),

                      "CORPORATE SOLUTIONS",
                      style: TextStyle(
                        color: Colors.white,
                        fontWeight: FontWeight.bold,

                    ),
                  ],
                ),

Description:

In this experiment, a visually structured Student ID Card user interface was designed and implemented using the Flutter framework on the FlutLab platform. The objective was to understand and apply fundamental Flutter widgets to build a clean, responsive, and aesthetically appealing UI layout.

Special attention was given to layout alignment, spacing, and hierarchy to ensure clarity and readability of information. The use of colors, typography, and structured divisions (such as headers and separators) helped in creating a professional UI design similar to real-world ID cards.

Through this implementation, the experiment demonstrates how Flutter enables rapid UI development with reusable components while maintaining consistency and flexibility across different screen sizes.
